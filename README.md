# Sistema de Gestión de Tickets de Soporte TI

Sistema Full Stack para la gestión de tickets de soporte técnico, desarrollado con React, Node.js, Express y MySQL.

## Tecnologías utilizadas

- React
- Node.js
- Express
- MySQL
- JWT
- Axios
- React Router
- Postman
- Git/GitHub

## Funcionalidades

- Registro e inicio de sesión de usuarios
- Autenticación con JWT
- Protección de rutas
- Roles: admin, técnico y usuario
- Creación de tickets
- Listado de tickets
- Detalle de ticket
- Comentarios por ticket
- Asignación de técnico
- Cambio de estado
- Filtros por estado y prioridad
- Búsqueda de tickets
- Paginación
- Dashboard frontend
- Modal para crear tickets
- Logout
